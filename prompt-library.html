Weekly summary from raw log data

Here is a list of [escalations/tickets/incidents] from this week [paste data]. Group them by root cause, identify the top 3 most frequent categories, and note any category that increased compared to last week. Format as a short bulleted summary for a team update.
Reporting

Variance report narrative

Here is this month's budget vs actual data by category [paste table]. Write a short narrative (under 200 words) explaining the three largest variances in plain business language, suitable for a management summary. Note where further investigation may be warranted.

Write a SQL query that [describe what you need, e.g. "finds customers who ordered in the last 30 days but had no orders in the prior 90 days"]. Use CTEs for readability, add comments explaining each step, and note any assumptions about the schema.
